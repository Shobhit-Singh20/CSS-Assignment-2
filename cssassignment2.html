Combinators

This is for Descendant Selector.

The descendant selector matches all elements that are descendants of a specified element.

The styling isn't applied on this.

This is for Child Selector(>).

The child selector selects all elements that are the children of a specified element.

The styling isn't applied on this.

This is for Adjacent Sibling Selector(+).

The adjacent sibling selector is used to select an element that is directly after another specific element.

This is a paragraph which is just outside the div block.

But this is not the immediate one.

This is for General Sibling Selector(~).

The general sibling selector selects all elements that are next siblings of a specified element.

This is the block which is outside the div block.

This is the another div block.

So all of them are siblings of division and therefore have the styling.
